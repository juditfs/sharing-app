# Sharene Prototype

Sharene is a **privacy-first photo sharing app** that lets you share photos via secure, time-limited links. It features client-side encryption, EXIF stripping, and a secure serverless backend.

## ðŸš€ Quick Start

### 1. Prerequisites
- **Node.js** (v18+)
- **Supabase CLI** (`brew install supabase/tap/supabase`)
- **Expo Go** on your iOS device (or Xcode Simulator)

### 2. Backend Setup
The backend is powered by Supabase (PostgreSQL + Edge Functions).

```bash
# Login and link project (if not done)
supabase login
supabase link --project-ref <your-project-ref>

# Deploy database schema & security policies
supabase db push

# Deploy Edge Functions
supabase functions deploy create-link --no-verify-jwt
supabase functions deploy get-link --no-verify-jwt
```

### 3. Run the Mobile App
The mobile app is built with React Native & Expo.

```bash
cd mobile
npm install
npx expo start
```
*   Press `i` to run in iOS Simulator.
*   Scan QR code with Expo Go on physical device.

### 4. Run the Web Viewer
The viewer is a Next.js app for recipients to view shared links.

```bash
cd viewer
npm install
npm run dev
```
*   Open `http://localhost:3000` (Note: Links generated by the app will point to your production URL, you may need to manually test with localhost if developing locally).
